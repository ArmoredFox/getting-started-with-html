The Basics of HTML

HTML stands for Hypertext Markup Language, and the internet is basically HTML documents linked
together for you to interact with. It has some basic elements, the text, which is the content you see,
the markup which is what it looks like, and then links and references to other documents.
HTML is made of elements and tags, where in you tell the computer what element to use
by using the proper tag to talk to it.

Examples of HTML

Basic commands are executed by using open tags and closed tags.
An open tag may look like < sample> and a closed tag like < /sample>.
Not all commands require a closed tag, some are void tags like < img > or < br >.

Inline Vs Block

There are two kinds of elements, inline and block, and they effect the page differently.
Inline elements exist how they sound, in the line. Block elements create an 'invisible box' around the element.

Dumb Computers

One thing that is most important to rememeber about coding is that computers may be able to do alot
of things, but thinking is not one of them. Computers are stupid. They read exactly what you have written
and nothing else, if you made a mistake in the code the computer isn't going to fix it, its going to run
with it and do things you don't want it to.
